# 🧠 Simulador de Decisiones y Evaluación de Sistemas de Inteligencia Artificial

## 📌 Descripción del Proyecto

Este repositorio contiene un **simulador interactivo de toma de decisiones en sistemas de Inteligencia Artificial**, acompañado de un **módulo de evaluación de sesgos y perfilado de IA**.

El objetivo del proyecto es demostrar, de forma práctica, cómo las decisiones en el diseño de sistemas inteligentes afectan aspectos clave como:

- ⚖️ Ética  
- 📊 Sesgos  
- 🔍 Transparencia  
- 🌍 Impacto social  

Además, se incluyen materiales teóricos en formato presentación que respaldan los fundamentos del proyecto.

---

## 🚀 Funcionalidades

### 🔹 1. Simulador de decisiones
- Escenarios reales (20 casos)
- Toma de decisiones interactiva
- Evaluación inmediata de consecuencias
- Sistema de puntuación ética

---

### 🔹 2. Evaluador y perfilador de IA
- Análisis basado en 6 variables críticas:
  - Datos
  - Diversidad
  - Supervisión
  - Transparencia
  - Impacto social
  - Seguridad
- Generación de perfil del sistema
- Diagnóstico automático
- Visualización mediante gráfico
